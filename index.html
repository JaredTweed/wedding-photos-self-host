Shared Lens

Different perspectives, one story.

Shared Lens now runs as a fully self-hosted gallery. There is no Firebase, no Google sign-in, no Stripe, and no cloud storage dependency at runtime. The admin side is protected with a local password, and uploads are stored on disk inside the Docker volume.
